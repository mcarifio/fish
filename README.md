# fish configuration directory

Usage:

```bash
sudo apt-get update
sudo apt-get install -y fish
chsh -s /usr/bin/fish

# logout and log back in

git clone git@github.com:mcarifio/fish.git ~/.config/fish
emacs ~/.config/fish ## add your stuff
```

Layout:


```
    ~/.config/fish
├── bin
├── config.fish
├── functions
│   ├── example.fish
└── README.md
```
